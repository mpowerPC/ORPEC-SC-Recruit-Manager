## Synopsis
A Discord bot to manage ORPEC's member database.

## File Structure
```
ORPEC-SC-Recruit-Manager
|-- resources
    |-- client_secret.json
    |-- token.txt
    |-- support.py
|-- bot.py
```

* client_secret.json: Generate @ https://console.developers.google.com/apis/credentials.
* token.txt: plain text of Discord API Token.
